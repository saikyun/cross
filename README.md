# cross

![image](https://user-images.githubusercontent.com/2477927/160694091-defaac53-85df-4982-9077-fe5478af9b09.png)

2d top down action game made in 3h.

+15 min for small fix.

## dependencies

* janet
* jpm

## how to run

```
git clone https://github.com/saikyun/cross
cd cross
jpm deps
janet cross.janet
```
